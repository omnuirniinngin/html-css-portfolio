Task Assigning and Planning

A mobile application for Task Assigning that is efficient, effective, and convenient
for users and is concerned for monitoring and controlling the project development process.

Context Diagram

In this context diagram gives an overview on how the entire system is built. The researchers identified the external entities which are Project Managers and Members that can be interchangeable since both entities can use all the functionalities. Creating a plan is available for both users since anyone could start a project plan on their own. After creating a plan follows with creating activities that can be also called as task where user can assign task by searching and adding a team member onto a plan. Monitoring and controlling tasks progress can only be done when a team member successfully accepted the pending requested by his project manager. Thus, Monitoring and Controlling can be achieved when assigned member inputs and update a tasks trough adjusting progress bar than can also be view to percentage and other necessary information such as notes.
